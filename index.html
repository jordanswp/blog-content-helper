Blog Upload Helper

This tool is designed to streamline the blog upload process by effortlessly removing extraneous header texts from HTML content.

The application removes the header texts (e.g. H2, H3 H4) from a block of HTML content, removes extra '</strong>' tags and changes the '<p>' tags to the appropriate header tags.
